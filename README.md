# Kelime Hatırlatıcı

Android için basit bir kelime öğrenme ve tekrar uygulaması.

## Özellikler

- Kelime ekleme
- Hazır İngilizce kelime paketleri
- Günlük hedef takibi
- Quiz modu
- Yanlış yapılan kelimeleri listeleme
- CSV, Excel ve Lingoes TXT içe aktarma
- Text-to-Speech ile kelime seslendirme
- GitHub Actions ile APK oluşturma

## APK Oluşturma

GitHub reposunda:

1. Actions sekmesine gir.
2. Android APK Build workflow'unu çalıştır.
3. Artifacts bölümünden APK dosyasını indir.
